# Behavior Tools SaaS

## Setup
1. Copy .env.example to .env
2. Add your Supabase keys
3. Run npm install
4. Run npm run dev

## Features
- Auth (ready for Supabase)
- Database (habits table)
- SaaS dashboard UI
- AI coaching scaffold

## Deploy
Use Vercel and add environment variables.
